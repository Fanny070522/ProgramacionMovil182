¿qué es Java Script?
Es uno de los lenguajes de programación más populares y quizá el más utilizado
trabajos como: Desarrollo web 
	(Frontend- parte visual de la aplicación)
	(backend- escribe la pare lógica)
	(Fullstack- desarrollador de ambas partes como de front y back)
Y son los encargados de construir las app móviles para Android e iOS y sobre microcontroladores así como Javascript es la navaja suiza de los lenjuages de programación 

Frameworks: Entrega herramientas y muchas más herramientas para desarrollar aplicaciones que sean complejas. Ejemplos: Veu, Angular, React (más popular dentro de JS)
Librerías: Encargado de solucionar un problema específico y tambien son herramientas 
El framework entrega más herramientas que una librería y se puede construir una aplicación completa utilizando solamente herramientas de framework, y una librería solo intenta resolver un problema en específico.

Historia de JS: 
Se hizo para ejecutarse dentro de los exploradores web especificamente dentro de netscape (la razón de su creación es por que los sitios web eran estáticos) 
Ecmascript (european computer manufacturers assosiation): repreenta una organización con el fin de contribuir a estándares tecnológicos. Ejemplo: ECMA 334 (especificaciones técnicas de c sharp), ECMA 404 (especificaciones de Json), ECMA 408 (especificaciones técnicas del lenguaje Dart), ECMA 262 o ECMascript (estándares para los lenguajes de programación js, action script) y la versión del mismo en el año 2015 se introdujeron funcionalidades como por ejemplo: LED const, fat Arrow functions, clases en js, etc)
Ecmascript 2015 cambio el mundo de javascript y js fue un lenguaje que se escribio para los exploradores web y todos tienen un motor que se encarga de ejecutar JS (javascrit engine o motor javascript) y el motor de Google Chrome es V8 siendo de código abierto y cualquier persona en todo el mundo puede acceder a leer el código fuente.

El desarrollador Ryan d logró enbeber este motor de V8 dentro de código escrito en C++, permitiendo que el código de JS pueda ser ejecutado practicamente en cualquier lado gracias a este señor. y pueda ser ejecutado en servidores, aplicaciones de ecritorio, móviles, videojuegos o microcontroladores. El código escrito en C++ permitió enbeber el motor V8 (node js).
